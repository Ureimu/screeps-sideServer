# 虫洞生成规则

1. 路口生成

路口生成的虫洞不会过期，没有unstableDate或decayTime属性，只会生成目的地为邻近shard房间的虫洞。

2. 区块中央房间生成

区块中央房间生成的虫洞会过期，会有unstableDate或decayTime属性，只会生成目的地为本shard房间的虫洞。

3. 已关闭区块周围生成

已关闭区块周围生成的虫洞不会过期，没有unstableDate或decayTime属性，只会生成目的地为已关闭区块周围的虫洞。主要用于快速通过已关闭区块。
